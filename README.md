## Description
Amazing speech to text app built using react js

## Tech-Stacks Used
 - React.js
 - CSS
 - HTML

## Setting up Locally 


```
1. Fork the repo
```
```
2. git clone "https://github.com/[Your_GithubUserName]/Speech-to-text.git"
```
```
3. cd Speech-to-text
``` 
```
4. npm install 
``` 
```
5. npm start
```
